# 主攻案例實作規格

## 主攻案例

- 題目: 黑潮分支流進入南海與臺灣海峽的流場類比
- 主要參考文獻: [Han et al. 2021](https://doi.org/10.3390/jmse9121465)
- 類型: 類比研究，不宣稱重現 ENSO 年事件或漁獲統計結果

## 科學問題

在 ODB SADCP 公開 0.25 度格點流場下，能否辨識出支持 Kuroshio Branch Current 向西北侵入南海北部與臺灣海峽的典型背景流場結構，並比較不同季節或月平均條件下該結構的強弱差異？

## 目標輸出

本輪實作應至少交付以下三個輸出。

## Output A: 主圖

- 區域: `lon=119.5~123.5`, `lat=20.5~26.5`
- 背景: 20-300 m 平均流速色階散點
- 向量: 25-35 m 平均流向箭頭
- 模式:
  - 年平均 `mode=0`
  - 東北季風 `mode=17`
  - 西南季風 `mode=18`
- 目的:
  - 呈現黑潮主軸
  - 觀察呂宋海峽以西是否出現較明顯的西北向入侵流場

## Output B: 季風比較雙子圖

- 左圖: `mode=17`
- 右圖: `mode=18`
- 其餘參數與 Output A 主圖一致
- 目的:
  - 比較季風背景下 KBC 類型流場是否更易出現

## Output C: KBC Proxy 圖或表

定義一個簡單且可重算的 proxy，二選一：

1. 固定斷面法
   - 斷面建議: `121.0E` 到 `121.5E` 間、`20.75N~23.0N`
   - 指標: 指定盒內所有格點中，具有西向分量 `u < 0` 且北向分量 `v > 0` 的格點比例，或其平均速度大小

2. 目標區域法
   - 盒區建議: `120.5E~121.75E`, `21.0N~23.5N`
   - 指標: 區域內「西北向流」格點比例
   - 西北向條件: `u < 0`, `v > 0`, 且方向角落在 `270~360` 度對應之西北象限定義下的子區間

建議先用第 2 種方法，因為它對 0.25 度格點較穩健。

## API 資料流程

## SADCP 背景流速

- Endpoint: `https://ecodata.odb.ntu.edu.tw/api/sadcp`
- 參數:
  - `lon0=119.5`
  - `lon1=123.5`
  - `lat0=20.5`
  - `lat1=26.5`
  - `dep0=20`
  - `dep1=300`
  - `dep_mode=mean`
  - `mode=0` 或 `17` 或 `18`
  - `append=u,v,speed,direction`

## SADCP 表層向量

- Endpoint 同上
- 參數:
  - `dep0=25`
  - `dep1=35`
  - `dep_mode=mean`
  - `mode=0` 或 `17` 或 `18`
  - `append=u,v,speed,direction`

## 選配 GEBCO 背景

- 只作地形輔助，不作主驗證依據
- `sample=1`
- 預設 `tile_deg=2`

## 視覺化構圖

- 主要風格沿用第二版成熟畫風：
  - GEBCO relief 可作淡背景
  - 海流色階點維持 `jet` 或同等藍黃紅高辨識色盤
  - 白色箭頭顯示 25-35 m 表層流向
- 主圖不要塞過多文字。
- 圖說應明寫：
  - 深層平均流速代表背景結構
  - 表層箭頭代表近表層流向
  - 這是 KBC 條件的類比圖，不是事件年重建

## 何者是重現，何者是類比

## 可視為重現的部分

- 研究問題的地理核心:
  - 黑潮是否向西北侵入南海北部與海峽
- 研究主題的物理背景:
  - KBC 路徑與背景流場

## 只能視為類比的部分

- ENSO 年門檻
- 漁獲統計連動
- 原論文中的 50% / 70% 入侵比例門檻
- 任何具年份與事件性的輸送結論

## 驗證方法

1. 文獻對照
   - 圖說與 README 必須明確引用 [Han et al. 2021](https://doi.org/10.3390/jmse9121465)
   - 只對應到「KBC 侵入可能改變輸送路徑」這一層級，不對應其全部結論

2. 場型驗證
   - 年平均圖應能看出臺灣東側黑潮主軸
   - 東北季風與西南季風圖應呈現可辨識差異
   - KBC proxy 應能隨模式改變而變化

3. 技術驗證
   - 資料流程只使用公開 API
   - 不引入需要授權的原始資料

## 失敗條件

以下任一情況成立，即視為主攻圖失敗：

- 季風或月平均圖在目標區域內無可讀差異，無法支持任何 KBC 類比判讀。
- proxy 在不同模式下毫無辨識度，無法形成分析問題。
- GEBCO 或視覺強化掩蓋了海流判讀。

## 替代方案

若主攻圖失敗，依序切換：

1. 改為「東臺灣黑潮平均結構」方法型案例，做更穩定的結構圖與深淺層對照。
2. 若仍要保留南海敘事，改為 SAE 題材的背景流場類比，但禁止宣稱已辨識 SAE 本體。
